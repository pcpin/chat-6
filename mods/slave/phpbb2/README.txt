===============================================================================
 This is an interface module between PCPIN Chat 6 and phpBB2 software.
 It allows you to use PCPIN Chat 6 as an addon to phpBB2 software.
===============================================================================



REQUIREMENTS
-------------

   1. phpBB version 2.0.22
   2. PCPIN Chat version 6.1




ACTIVATION
-----------

   Log in as administrator and open Admin Panel. In the Navigation frame
   select "Settings" -> "Slave mode". Slave mode configuration will appear.
   Select "phpbb2" from "Slave mode master" list, set "Activate slave mode" to "Yes"
   and click "Save changes". Slave mode is now active.




SUPPORT AND TROUBLESHOOTING
----------------------------

   If you have any problems installing or using this module, then you can
   get help in PCPIN Community Forums: http://community.pcpin.com/
